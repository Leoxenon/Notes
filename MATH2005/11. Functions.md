- Polynomials
	- degree
		- deg(P(x))
	- Let P(x) and Q(x) be any two polynomials in x
		- $deg(P(x)±Q(x)) ≤ max(deg(P(x)),deg(Q(x)))$
		- $deg(P(x)Q(x)) = deg(P(x)) +deg(Q(x))$
	- $P(x) ≡ Q(x)$
		- P(x) is identically equal to Q(x)

identity of polynomials
```
syms x
p(x) = x^3 + 1 
q(x) = (x+1)*(x^2-x+1) 
isAlways(p(x) == q(x))
solve(x^2 + 2 == 3*x)
```
Solve coefficients by identity
```
syms p q m n
eqn1 = p - 2 == -1 
eqn2 = q - 2*p + 3 == -3
eqn3 = 3*p - 2*q == m
eqn4 = 3*q == n 
sol = solve ([eqn1, eqn2, eqn3, eqn4], [p,q,m,n]) 
sol.m 
sol.n
```

- Algebaric fraction
	- $\frac{P(x)}{Q(x)}$
	- proper fraction
		- $deg(P(x)) < deg(Q(x))$
	- improper fraction
		- $deg(P(x)) ≥ deg(Q(x))$
- Rational Functions
	-  $f(x) = \frac{P(x)}{Q(x)}$
